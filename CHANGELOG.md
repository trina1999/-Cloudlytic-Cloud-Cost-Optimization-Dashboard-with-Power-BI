# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-13

### Added
- Initial release of Cloudlytic
- Azure pricing data collector
- AWS pricing data collector
- Data processor for normalizing pricing data
- Cost optimizer with savings identification
- Cost forecasting using linear regression
- Sample data generator for testing
- Power BI Dashboard Build Guide
- Complete DAX measures library
- Setup and user documentation
- Support for Compute, Storage, and Network categories
- Bill of Materials tracking
- 6 comprehensive dashboard pages:
  - Executive Summary
  - Cost Comparison
  - Usage Trends
  - Cost Optimization
  - Cost Forecast
  - Bill of Materials

### Documentation
- README with quick start guide
- Setup guide with troubleshooting
- User guide with dashboard features
- Dashboard build guide with step-by-step instructions
- Contributing guidelines
- MIT License
